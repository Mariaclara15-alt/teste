Todas as regras da cascata

Regras da Cascata: algoritmo que resolve conflitos quando múltiplas regras de estilo se aplicam ao mesmo elemento HTML, determinando qual delas tem prioridade para definir a aparência final.

A cascata vai determinar que tudo que seja escrito vai ser lido de cima para baixo

E se houver problemas de conflitos de propriedade e selecinar h1, a última a ser selecionado vai prevalecer.

A cascata ajuda a resolver certos conflitos dentro do CSS entre eles estão:

  1. Origem de estilo
  2. Especificidade
  3. Uma ordem de declaração

Combinadores

Combinador é quando eu coloco um seletor junto com outro seletor, ou seja qualquer h1 que estiver na div primeira vai ser selecionado.

  • Evita repetir classes nos IDs
  • Deixa mais simplificado a organização do CSS
  • Deixa o CSS com um aspecto mais organizado e nada muito poluído
  • Permite aplicar diversas coisas
